# Ivan Dachev Website

## Local

```bash
bundle exec jekyll serve
```

Open [http://localhost:4000](http://localhost:4000) in your browser.

```bash
utils/build-site.sh
```

## Deploy

```bash
utils/deploy-site.sh
```
